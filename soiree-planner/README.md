# Soiree Planner

This website is a static HTML, CSS, and JavaScript project, so it does not require installation or a build step.

## Open locally

Open `index.html` in a web browser. On Windows, you can double-click the file in File Explorer or run this command from the `soiree-planner` folder:

```powershell
Start-Process .\index.html
```

The page uses Google Fonts, so an internet connection is needed for the intended fonts to load.
